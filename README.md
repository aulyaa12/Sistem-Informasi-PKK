# Sistem Informasi Data PKK Desa

Sistem Informasi Data PKK Desa adalah aplikasi berbasis web yang dibuat untuk membantu pengelolaan data PKK desa secara lebih rapi, mudah, dan terstruktur.

Aplikasi ini menyediakan beberapa modul utama untuk mencatat dan mengelola data warga desa, seperti data penduduk, data kelahiran, data kematian, dan data lansia.

## Fitur Utama

- Mengelola data penduduk desa
- Mencatat data kelahiran warga
- Mencatat data kematian warga
- Mengelola data lansia
- Menampilkan dashboard statistik
- Mengunduh rekap data dalam bentuk Excel
- Mendukung tampilan responsif

## Modul Sistem

Sistem ini memiliki empat modul utama:

1. **Data Penduduk**  
   Digunakan untuk mengelola data dasar warga desa.

2. **Data Kelahiran**  
   Digunakan untuk mencatat data kelahiran warga desa.

3. **Data Kematian**  
   Digunakan untuk mencatat laporan kematian warga desa.

4. **Data Lansia**  
   Digunakan untuk memantau data warga lanjut usia.

## Alur Penggunaan

1. Pengguna melakukan pendaftaran akun.
2. Admin melakukan verifikasi akun.
3. Pengguna masuk ke sistem.
4. Pengguna mengelola data sesuai wilayahnya.
5. Data dapat dilihat dalam bentuk tabel, grafik, dan file Excel.

## Teknologi yang Digunakan

- PHP
- CodeIgniter 4
- MySQL
- Bootstrap 5
- Chart.js
- JavaScript

## Tampilan Sistem

Sistem memiliki tampilan sederhana dan responsif, sehingga dapat digunakan melalui laptop maupun perangkat mobile.

## Tujuan Pengembangan

Project ini dibuat untuk membantu digitalisasi pengelolaan data PKK desa agar proses pencatatan, pemantauan, dan rekap data menjadi lebih mudah.
