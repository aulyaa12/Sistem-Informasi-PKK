# Sistem Informasi Data PKK Desa

Sistem Informasi Data PKK Desa adalah aplikasi berbasis web yang dikembangkan untuk membantu proses pengelolaan data PKK desa secara lebih rapi, mudah, dan terstruktur. Aplikasi ini digunakan untuk mencatat, mengelola, memantau, dan merekap data warga desa yang berkaitan dengan kegiatan administrasi PKK.

## Deskripsi Sistem

Aplikasi ini menyediakan beberapa modul utama untuk mendukung pengelolaan data desa, seperti data penduduk, data kelahiran, data kematian, dan data lansia. Selain itu, sistem juga dilengkapi dengan dashboard statistik dan fitur ekspor data ke dalam format Excel, sehingga memudahkan pengguna dalam melihat ringkasan data dan membuat laporan.

## Fitur Utama

* Mengelola data penduduk desa
* Mencatat data kelahiran warga
* Mencatat data kematian warga
* Mengelola data lansia
* Menampilkan dashboard statistik
* Mengunduh rekap data dalam format Excel
* Mendukung tampilan responsif untuk perangkat desktop dan mobile
* Mengatur akun pengguna melalui proses pendaftaran dan verifikasi admin

## Modul Sistem

### 1. Data Penduduk

Modul ini digunakan untuk mengelola data dasar penduduk desa, seperti identitas warga, nomor kartu keluarga, jenis kelamin, alamat, RT, pekerjaan, pendidikan, dan status pernikahan.

### 2. Data Kelahiran

Modul ini digunakan untuk mencatat data kelahiran warga desa, seperti nama bayi, jenis kelamin, tanggal lahir, tempat lahir, data ibu, dan keterangan tambahan.

### 3. Data Kematian

Modul ini digunakan untuk mencatat data kematian penduduk desa, seperti identitas almarhum, tanggal kematian, tempat kematian, dan data penduduk terkait.

### 4. Data Lansia

Modul ini digunakan untuk mengelola data warga lanjut usia, termasuk informasi umur, hobi, keterampilan, produktivitas, dan keterangan lainnya.

## Alur Penggunaan Sistem

1. Pengguna melakukan pendaftaran akun.
2. Admin melakukan verifikasi terhadap akun pengguna.
3. Pengguna yang telah disetujui dapat masuk ke dalam sistem.
4. Pengguna mengelola data sesuai desa atau wilayahnya.
5. Data dapat dilihat dalam bentuk tabel, grafik, dan file Excel.
6. Pengguna dapat mengunduh rekap data sesuai kebutuhan laporan.

## Teknologi yang Digunakan

* PHP
* CodeIgniter 4
* MySQL
* Bootstrap 5
* Chart.js
* JavaScript

## Tampilan Sistem

Sistem dirancang dengan tampilan sederhana, responsif, dan mudah digunakan. Dengan dukungan Bootstrap 5, aplikasi dapat diakses melalui perangkat desktop maupun mobile.

## Tujuan Pengembangan

Project ini dikembangkan untuk membantu digitalisasi pengelolaan data PKK desa. Dengan adanya sistem ini, proses pencatatan, pemantauan, pencarian, dan rekap data dapat dilakukan secara lebih cepat, terstruktur, dan efisien dibandingkan pencatatan manual.

## Kesimpulan

Sistem Informasi Data PKK Desa diharapkan dapat menjadi solusi digital dalam pengelolaan data administrasi PKK desa. Aplikasi ini membantu pengguna dalam mengelola data penduduk, kelahiran, kematian, dan lansia secara terpusat serta mendukung penyusunan laporan melalui fitur dashboard dan ekspor data.
